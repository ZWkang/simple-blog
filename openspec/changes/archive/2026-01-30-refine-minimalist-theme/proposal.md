# 变更：打造极简主题风格

## 为什么

当前博客使用 Nextra 的默认主题，并有一些自定义样式（如 h1 渐变标题），但缺乏连贯的极简主义设计理念。精心打造的极简主题将提升可读性，减少视觉干扰，创造更专注的阅读体验，让内容本身成为焦点而非装饰。

## 改变什么

- **排版优化**：建立清晰的排版层级，为长文阅读优化行高、字号和间距
- **色彩简化**：定义极简调色板，专注于高对比度和可读性
- **布局优化**：实现慷慨的留白、最佳行长（45-75 字符）和清晰的内容边界
- **组件样式**：简化 UI 组件（导航栏、页脚、代码块、提示框）以符合极简原则
- **视觉一致性**：移除或优化装饰性元素（如渐变 h1）以保持视觉和谐
- **深色模式优化**：确保深色模式遵循极简原则，具有适当的对比度

## 影响范围

- **受影响的规范**：
  - `visual-theme`（新能力）
  - `typography`（新能力）
  - `layout`（新能力）

- **受影响的代码**：
  - `app/layout.jsx` - 主题配置和布局结构
  - `mdx-components.jsx` - 组件样式覆盖
  - `next.config.js` - Nextra 主题配置
  - 新增 CSS/样式文件用于自定义主题实现

- **用户体验**：提升可读性和内容专注度，更清爽的视觉呈现

- **破坏性变更**：无 - 纯视觉优化，保持所有现有功能
